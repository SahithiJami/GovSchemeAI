# GovScheme AI

## AI-Based Government Scheme Awareness & Eligibility Analyzer

GovScheme AI is a web application developed using Python, Streamlit, Data Analytics, and Artificial Intelligence to help citizens discover government schemes, check eligibility, and receive AI-powered recommendations.

## Features

- Dashboard with analytics
- Government Scheme Eligibility Checker
- Community Survey Collection
- Survey Data Analytics
- AI-Based Scheme Recommendation
- OpenRouter AI Assistant
- Download Survey Data as CSV

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly
- OpenRouter API
- GitHub

## Project Domain

Data Analytics for Community Development

## Objective

The project aims to increase awareness of government welfare schemes and support community development through data-driven insights and AI-powered assistance.
